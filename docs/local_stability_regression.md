# 本地稳定使用版回归清单

> 目标：本地稳定使用，不包含 SQLite 迁移、跨平台安装包、AI 功能。

---

## 每轮开发后必跑命令

```powershell
git status
pnpm test
pnpm build
```

---

## 一、文件闭环回归

| # | 验收项 | 步骤 | 预期 |
|---|--------|------|------|
| 1 | 新建文件 | Ctrl+N → 输入内容 | 编辑器进入空白状态 |
| 2 | 打开文件 | Ctrl+O → 选择 `.md` 文件 | 内容正确显示 |
| 3 | 保存 | Ctrl+S | dirty 状态清除，文件写入磁盘 |
| 4 | 另存为 | Ctrl+Shift+S → 选择新路径 | 新文件创建，原文件不变 |
| 5 | 关闭未保存确认 | 修改后 Ctrl+W / 打开新文件 | 弹出保存/放弃/取消三个选项 |
| 6 | 保存并退出 | 确认弹窗选择"保存" | 保存成功后继续操作 |
| 7 | 放弃更改 | 确认弹窗选择"放弃" | 不保存继续操作 |
| 8 | 取消关闭 | 确认弹窗选择"取消" | 回到编辑器，内容不变 |
| 9 | 自动保存 | 打开已保存文件 → 修改 → 等待 debounce | 自动保存生效（状态栏显示） |
| 10 | 中文路径文件 | 打开含中文路径的 `.md` 文件 | 正常打开和保存 |
| 11 | 重新打开 | 保存后关闭 → Ctrl+O 再次打开 | 内容不变，格式不损坏 |

## 二、图片链路回归

| # | 验收项 | 步骤 | 预期 |
|---|--------|------|------|
| 1 | 已保存文档拖拽图片 | 打开已有 `.md` → 拖入 png/jpg | 图片复制到 `.assets`，插入相对路径 |

| 3 | 粘贴图片 | 已保存文档 → Ctrl+V 粘贴剪贴板图片 | 写入 `.assets`，插入相对路径 |
| 4 | 相对路径正确 | 查看 Markdown 源码 | 路径格式 `./xxx.assets/image-xxx.png` |
| 5 | 写作模式图片显示 | 写作模式查看带图片文档 | 图片正常渲染 |
| 6 | 源码模式图片语法 | 源码模式查看带图片文档 | 显示 `![alt](./xxx.assets/xxx.png)` |
| 7 | 分屏模式图片一致 | 分屏模式查看带图片文档 | 预览区图片正常 |
| 8 | 保存后重开 | Ctrl+S → 关闭 → 重新打开 | 图片仍显示 |
| 9 | 导出 HTML 图片正常 | 导出 HTML → 浏览器打开 | 图片显示（base64 内嵌或相对路径可用） |

## 九、不通过处理规则

| 优先级 | 场景 | 处理 |
|:---:|------|------|
| 🔴 | 保存/另存为/文件读写失败 | **停止继续开发**，必须先修复 |
| 🔴 | 图片路径损坏或相对路径失效 | **停止继续开发**，必须先修复 |
| 🔴 | 关闭未保存确认绕过 | **停止继续开发**，必须先修复 |
| 🔴 | `pnpm test` 或 `pnpm build` 失败 | **必须先修复** |
| 🟡 | UI 小问题（样式偏移、颜色异常） | 记录后继续，后续统一修复 |
| 🟡 | 导出 HTML/PDF 样式轻微偏差 | 记录后继续 |
| 🟢 | Mermaid/KaTeX 渲染偶发失败 | 记录后继续（这些库本身有版本兼容性） |
